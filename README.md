# Remaining Time

A simple partially vibe-coded script to calculate remaining time of Wplace paint charges refill.

## Create Standalone Executable (.exe)

To create a standalone `.exe` file, follow these steps:

1.  **Install PyInstaller**:
    ```bash
    pip install pyinstaller
    ```

2.  **Build the Executable**:
    Run the following command in your terminal:
    ```bash
    pyinstaller --onefile main.py
    ```

3.  **Locate the File**:
    The generated `.exe` file will be in the `dist` folder.
